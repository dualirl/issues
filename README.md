# dualirl — Issues

Bug reports and feature requests for the dual-camera IRL streaming app.

### Quick links
- 🐛 [**Report a bug**](https://github.com/dualirl/issues/issues/new?template=bug_report.md)
- ✨ [**Request a feature**](https://github.com/dualirl/issues/issues/new?template=feature_request.md)
- 🔎 [**Browse all issues**](https://github.com/dualirl/issues/issues?q=is%3Aissue)

## Before you post
- **[Search existing issues](https://github.com/dualirl/issues/issues?q=is%3Aissue)** first — yours may already be reported (add a 👍 instead of filing a duplicate).
- **Never paste secrets.** Scrub stream keys, SRT passphrases, and server URLs/tokens out of any logs or screenshots before posting.

## Filing a good report
Pick the right template — [**Bug report**](https://github.com/dualirl/issues/issues/new?template=bug_report.md) or [**Feature request**](https://github.com/dualirl/issues/issues/new?template=feature_request.md) — and for bugs include your phone + Android version, the app version, and your stream setup (single/dual, protocol, codec).

Thanks for helping make the app better. 🎥
