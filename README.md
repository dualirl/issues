# dualirl — Issues

Bug reports and feature requests for the dual-camera IRL streaming app.

## Before you post
- **Search existing issues** first — yours may already be reported.
- **Never paste secrets.** Scrub stream keys, SRT passphrases, and server
  URLs/tokens out of any logs or screenshots before posting.

## Filing a good report
Use the bug template and include your phone + Android version, the app
version, and your stream setup (single/dual, protocol, codec).

Thanks for helping make the app better. 🎥
